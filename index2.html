Privacy Policy — Dev Wallet Flagger for Axiom

Last updated: November 2025

Introduction

Dev Wallet Flagger (“the Extension”) is a browser extension for Chromium based browsers. The Extension highlights recently-created developer wallets on Axiom.trade to help users identify newly launched tokens.

1. Data We Collect

1.1. Local Extension Settings (Stored on Your Device Only)

The Extension stores the following settings locally inside your browser using Chrome’s chrome.storage.local:

  • Flagging enabled/disabled
  • Age threshold setting

These values never leave your device.

1.2. Page Content Access

The Extension reads publicly visible information already shown on Axiom.trade pages, such as the creation age of a token’s developer wallet. This data is processed only in-memory and never stored or transmitted.

2. Data We Do NOT Collect

  • No browsing history
  • No wallet addresses
  • No personally identifiable information
  • No cookies
  • No account or trading info
  • No external API data
  • No data from non-Axiom.trade pages

The Extension does not send data to any remote servers or third parties.

3. How Your Data Is Used

All processing is local and temporary. The Extension uses detected wallet age only to determine whether to apply a visual highlight.

4. Third-Party Sharing

We do not share, sell, or transfer any data to third parties. The Extension uses no analytics, ads, tracking, or external APIs.

5. Permissions Explained

  • storage — Saves user settings locally.
  • host_permissions: https://axiom.trade/* — Allows reading wallet age on Axiom.trade pages.

6. Security

Data never leaves your device; settings are stored using Chrome’s secure extension storage.
